# 设计系统使用指南

## 📖 概述

本项目采用统一的设计系统，所有设计配置都集中在 `lib/design.config.ts` 中。通过修改这个配置文件，可以快速切换整个网站的风格，无需逐个修改组件。

## 🎯 设计系统架构

### 配置文件位置
- **主配置**: `lib/design.config.ts`
- **CSS 变量**: `app/globals.css` (基于配置生成)
- **设计规范**: `.cursor/rules/designrules.mdc`

## 🚀 快速使用

### 1. 导入配置

```typescript
import { 
  COLORS, 
  SPACING, 
  BORDER_RADIUS, 
  TRANSITIONS,
  getPrimaryClasses,
  getCTAClasses,
  getGlassClasses 
} from '@/lib/design.config';
```

### 2. 使用颜色

```tsx
// 方式一：使用 CSS 变量（推荐）
<div className="bg-[var(--primary)] text-[var(--text-primary)]">
  内容
</div>

// 方式二：使用配置对象
import { COLORS } from '@/lib/design.config';
<div style={{ backgroundColor: COLORS.primary.main }}>
  内容
</div>

// 方式三：使用工具函数
import { getPrimaryClasses } from '@/lib/design.config';
<button className={getPrimaryClasses().bg}>
  按钮
</button>
```

### 3. 使用间距

```tsx
import { SPACING } from '@/lib/design.config';

<div style={{ padding: SPACING.lg, margin: SPACING.md }}>
  内容
</div>
```

### 4. 使用玻璃效果

```tsx
import { getGlassClasses } from '@/lib/design.config';

// 基础玻璃效果
<div className={getGlassClasses().glass}>
  内容
</div>

// 玻璃卡片效果
<div className={getGlassClasses().glassCard}>
  内容
</div>
```

## 🔄 快速切换风格

### 场景：从暗色主题切换到亮色主题

1. **修改 `lib/design.config.ts`**:

```typescript
// 修改背景色配置
export const COLORS = {
  background: {
    main: '#F8FAFC',           // 改为浅色
    secondary: '#FFFFFF',       // 改为白色
    glass: 'rgba(255, 255, 255, 0.8)',  // 改为白色玻璃
    glassCard: 'rgba(255, 255, 255, 0.9)',
  },
  text: {
    primary: '#1E293B',         // 改为深色文本
    main: '#334155',
    // ... 其他颜色
  },
  // ... 其他配置
};
```

2. **更新 `app/globals.css` 中的 CSS 变量**:

```css
:root {
  --background: #F8FAFC;
  --text: #334155;
  /* ... 其他变量 */
}
```

3. **完成！** 所有使用配置的组件会自动更新。

## 📋 配置项说明

### COLORS - 颜色配置
- `primary`: 主色调
- `secondary`: 次色调
- `cta`: CTA 按钮颜色
- `background`: 背景色
- `text`: 文本颜色
- `border`: 边框颜色
- `status`: 状态颜色（成功、警告、错误、信息）
- `area`: 功能区域颜色（用于图表等）

### SPACING - 间距配置
- `xs`: 4px
- `sm`: 8px
- `md`: 16px
- `lg`: 24px
- `xl`: 32px
- `2xl`: 48px
- `3xl`: 64px

### BORDER_RADIUS - 圆角配置
- `sm`: 4px
- `md`: 8px
- `lg`: 12px
- `xl`: 16px
- `2xl`: 24px
- `full`: 完全圆形

### TRANSITIONS - 过渡动画配置
- `fast`: 150ms
- `normal`: 200ms
- `slow`: 300ms
- `easing`: 缓动函数

### TYPOGRAPHY - 字体配置
- `fontFamily`: 字体族
- `fontSize`: 字体大小
- `fontWeight`: 字体粗细
- `lineHeight`: 行高


## 📝 最佳实践

### ✅ 推荐做法

1. **使用 CSS 变量**
```tsx
<div className="bg-[var(--primary)]">
```

2. **使用配置对象**
```tsx
import { COLORS } from '@/lib/design.config';
<div style={{ color: COLORS.text.primary }}>
```

3. **使用工具函数**
```tsx
import { getPrimaryClasses } from '@/lib/design.config';
<button className={getPrimaryClasses().bg}>
```

### ❌ 避免做法

1. **不要硬编码颜色**
```tsx
// ❌ 错误
<div className="bg-[#2563EB]">

// ✅ 正确
<div className="bg-[var(--primary)]">
```

2. **不要硬编码间距**
```tsx
// ❌ 错误
<div style={{ padding: '24px' }}>

// ✅ 正确
import { SPACING } from '@/lib/design.config';
<div style={{ padding: SPACING.lg }}>
```

3. **不要创建重复的样式类**
```tsx
// ❌ 错误：在组件中定义样式
const styles = {
  primary: '#2563EB',
}

// ✅ 正确：使用配置
import { COLORS } from '@/lib/design.config';
```

## 🔧 维护和更新

### 添加新颜色
1. 在 `lib/design.config.ts` 的 `COLORS` 对象中添加
2. 在 `globals.css` 中添加对应的 CSS 变量（如果需要）
3. 更新本文档

### 修改现有颜色
1. 直接修改 `lib/design.config.ts`
2. 同步更新 `globals.css` 中的 CSS 变量
3. 所有使用该颜色的组件会自动更新

### 添加新风格方案
1. 创建新的配置对象
2. 提供切换函数
3. 更新文档说明
