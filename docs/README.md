# 项目文档目录

这个目录用于存放**重要的、必要的项目文档**。

## 📋 文档分类

根据项目文档规则，不同类型的文档应存放在不同位置：

- **`docs/`** (本目录) - 重要项目文档
  - 项目核心文档
  - 重要的技术规范
  - 项目架构说明
  - 关键配置文档
  - 必须保留的重要文档

- **`archive/docs/`** - 网站内容文档
  - 嵌入到网站中展示的 Markdown 文档
  - Git 学习教程和指南
  - 用户可以在网站上阅读的所有内容文档

- **`dev_docs/`** - 开发文档
  - 网站开发相关的指导文件
  - 开发计划文档
  - 开发过程中的笔记和记录
  - 临时文档和草稿

- **`content/git-docs/`** - Git 相关文档
  - Git 使用指南和教程

## ✅ 本目录特点

- ✅ 本目录下的所有 `.md` 文件**会被 Git 跟踪**
- ✅ 适合存放需要版本控制的重要文档
- ✅ 不会被 `.gitignore` 忽略

## 📝 使用建议

将以下类型的文档放在这里：

- 项目规范文档
- 核心 API 规范
- 重要技术决策文档
- 项目架构说明
- 关键配置说明
