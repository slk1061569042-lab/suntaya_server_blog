# Office Add-ins 演示说明

## 🎬 快速演示指南

### 演示 1：PPT 技术架构图生成器

**效果展示：**

1. **打开 PowerPoint**
2. **加载 Add-in**：插入 → 我的加载项 → 选择"技术架构图生成器"
3. **选择模板**：点击"微服务架构"模板卡片
4. **填写信息**：
   - 系统名称：电商平台
   - 组件：用户服务、订单服务、支付服务等
5. **生成架构图**：点击"✨ 生成架构图"按钮
6. **查看结果**：架构图自动插入到当前幻灯片

**生成效果：**
- 美观的 ASCII 架构图
- 自动布局和连接线
- 可编辑的文本框
- 支持多种架构模式

### 演示 2：PPT 演讲稿自动生成器

**效果展示：**

5. **勾选选项**：
   - ✅ 包含架构图
   - ✅ 包含代码示例
   - ✅ 包含演示步骤
6. **生成演讲稿**：点击"🚀 生成完整演讲稿"
7. **查看结果**：自动生成多张幻灯片，包括：
   - 封面页
   - 目录页
   - 项目背景
   - 技术架构图
   - 核心功能模块
   - 技术栈
   - 代码示例
   - 系统特点
   - 演示步骤
   - 总结

**智能特性：**
- 自动提取技术关键词
- 智能生成模块列表
- 自动生成架构图
- 根据时长调整幻灯片数量

### 演示 3：Excel 数据分析面板

**效果展示：**

1. **准备数据**：在 Excel 中准备一些数值数据（例如：销售数据、统计数据等）
2. **打开 Excel**
3. **加载 Add-in**：插入 → 我的加载项 → 选择"Excel 数据分析面板"
4. **选择数据范围**：
   - 在 Excel 中选择包含数据的单元格区域
   - 点击"📍 选择数据范围"按钮
5. **开始分析**：点击"🔍 开始分析"按钮
6. **查看结果**：
   - **统计卡片**：显示行数、列数、总和、平均值
   - **可视化图表**：自动生成柱状图
   - **数据预览**：显示前 10 行数据
7. **插入图表**：点击"📈 插入图表到 Excel"按钮
   - 图表自动插入到工作表中
   - 可调整位置和大小

**分析功能：**
- 实时数据统计
- 智能图表生成
- 数据预览
- 一键插入到 Excel

## 💻 技术实现亮点

### 1. PowerPoint API 使用

```javascript
// 添加文本框到幻灯片
const shape = slide.shapes.addTextBox(
    "文本内容",
    PowerPoint.ShapeTextOrientation.horizontal,
    50, 50, 600, 100  // x, y, width, height
);

// 设置文本样式
shape.textFrame.textRange.font.size = 24;
shape.textFrame.textRange.font.bold = true;
shape.textFrame.textRange.font.color = "0078D4";
```

### 2. Excel API 使用

```javascript
// 获取选中范围
const range = context.workbook.getSelectedRange();
range.load("values, rowCount, columnCount");
await context.sync();

// 创建图表
const chart = sheet.charts.add(
    Excel.ChartType.columnClustered,
    sheet.getRange(selectedRange),
    Excel.ChartSeriesBy.auto
);
```

### 3. 图表可视化

使用 Chart.js 库生成交互式图表：
- 柱状图
- 折线图
- 饼图（可扩展）
